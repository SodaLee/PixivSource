Linpx 书源（更新📆：2024-01-15）

可用功能：✅搜索✅发现✅添加网址✅订阅源  
搜索小说：✅单篇❌系列✅作者✅标签✅链接  
发现小说：✅推荐作者✅最近更新  
添加网址：✅Linpx 链接✅Linpx 分享链接  
搜索网址：✅Linpx 链接✅Pixiv 原文链接  
订阅用法：点击订阅源打开小说，点击【添加到书架】按钮添加小说到书架  
温馨提示：⚠️Linpx 搜索暂时不支持系列小说名称，添加系列小说后，无法再次通过搜索获取并更新目录
