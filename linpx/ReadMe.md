Linpx 书源（更新时间：2025-01-10）

可用功能：✅搜索✅发现✅添加网址✅订阅源
搜索小说：✅小说名称✅作者名称✅小说标签
发现小说：✅推荐作者✅最近更新
添加网址：✅Linpx小说链接(✅分享链接✅原链接)
订阅用法：点击订阅源打开小说，点击【添加到书架】按钮添加小说到书架
温馨提示：⚠️ Linpx 的系列小说 api 目前不会返回小说id，添加系列小说后，无法再次获取目录

书源发布：兽人阅读频道 https://t.me/FurryReading
